---
description: Enforce implementation guardrails for React Native mobile apps targeting both iOS and Android.
---

Use this skill when a project includes React Native.

Guardrails:
1. Keep domain logic out of presentational screens.
2. Respect navigation, state, and service boundaries.
3. Consider platform differences explicitly.
4. Protect offline/error/loading UX.
5. Be careful with permissions, deep links, and push notifications.
6. Call out native-module or build configuration impact.

Output:
- mobile structure review
- platform-specific concerns
- release/test impact
